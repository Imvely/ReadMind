오늘 작업을 마무리한다. CLAUDE.md §12.4 세션 종료 루틴을 따른다:
1. 완료 기준(§12.3)을 충족한 feature_list.json 항목만 passes:true 로 바꾼다.
   절반만 된 항목은 false 로 유지한다.
2. 변경사항을 git commit 한다 (메시지 규약 §9).
3. claude-progress.txt 에 아래를 구체적으로 추가한다:
   - [날짜] 오늘 완료한 것
   - 지금 막혀있는 것 (파일/줄 번호까지)
   - 다음에 제일 먼저 손댈 것
   - 참고할 컨텍스트(샘플 파일 경로, 임시 결정 등)
4. 모듈이 끝났으면 해당 README 와 명세서 §8 체크박스도 갱신한다.
